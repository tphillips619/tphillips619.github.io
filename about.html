About This Visualization

Narrative Visualization Choice

This visualization uses an Interactive Slideshow to implement its narrative. Each scene follows an author defined sequence and contains elements that interact with the viewer. Each scene has a Previous and Next button at the bottom of the page and a clearly selected order for the user to view the slides.

Use of Scenes

The visualization utilizes bootstrap.js to implement the pager buttons, navbar at the top, and for consistency with CSS. Elements (such as tooltips) that are not already specified in bootstrap are specified in the style section of the html header. Each page follows a consistent layout consisting of a navbar at the top, fixed size visualization underneath, title, paragraph, and pager buttons at the bottom.

Use of Annotations

The data used to generate each line graph contains between 220 and 2200 (x,y) pairs and interactive tooltips allow the viewer to investigate the line at each point. Tooltips for bar charts on the forecast page where done with mouseover events directly over the rectangle HTML objects. Tooltips for the line charts were more complicated and involved creating an invisible scatterplot of circles imposed over the SVG line. Each circle element was connected to the data and assigned a mouseover and mouseout event that changed the opacity of the HTML div that framed the tooltip.

Use of Parameters

The price history graph includes a legend click button to show or hide each line. A click changes the state of the line by toggling the opacity parameter which persists until the text is clicked again.

Use of Triggers

Mouseover and mouseout events trigger the show and hide functions for the tooltip. The price history graph includes an on click function that toggles the visibility of individual lines.

Data Sources

  • Price History: Zillow ZHVI 2-Bedroom Time Series ($)
  • Interest Rate: Freddie Mac 30-yr_historics.xls
  • Forecast Change: Zillow Home Value Forecast for all regions
