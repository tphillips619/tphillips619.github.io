Estimating Pi¶

In [1]:
import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

We know the formula for calculating the area of a circle is $\Pi r^2$. If we didn't know the value of $\Pi$, but know its formula, we can estimate it using a Monte Carlo method and repeatedly sample random numbers to generate observations that can be plugged back into our known formula.

In [2]:
num_observations = 100000

x = np.random.random(size=num_observations)
y = np.random.random(size=num_observations)

Given r=1, the area of a circle must be $\Pi$. When $x^2 + y^2 = 1$, the point would fall on the circumference of the circle. When $x^2 + y^2 < 1$, these values would fall within the area of the circle.

In [3]:
in_circle = np.square(x) + np.square(y) <= 1

When we count the number of points within the circle and divide this by total number of points, we can obtain our estimate, $\hat{\Pi}$

In [4]:
pi_hat = np.sum(in_circle)/num_observations
pi_hat
Out[4]:
0.78571

This number is not $\Pi$, and plotting x and y should help explain why.

In [5]:
plt.scatter(x,y,s=1, c=in_circle)
Out[5]:
<matplotlib.collections.PathCollection at 0x7fa78ed54cc0>

The range of values we used to generate our observations were between 0 and 1.

If we change the range of the points to numbers between -1 and 1, we should get $\hat{\Pi}$. We can reuse our x and y arrays and center data by subtracting 0.5. Then scale the data back so that r=1 and the points range between (-1,1) by multiplying it by 2.

In [6]:
x_quad = (x-0.5)*2
y_quad = (y-0.5)*2

in_circle = np.square(x_quad) + np.square(y_quad) <= 1
ax = plt.scatter(x_quad, y_quad, s=1, c=in_circle)
plt.axvline(x=0)
plt.axhline(y=0)
Out[6]:
<matplotlib.lines.Line2D at 0x7fa78e82f5c0>

This figure has a radius of 1, an area of $\Pi$, and explains what we were estimating in the first plot. When we used numbers between (0,1), we were only estimating observations in Quadrant 1.

In [7]:
pi_hat*4
Out[7]:
3.14284

When multiplying the the result by 4, we obtain $\hat{\Pi}$ plus or minus error ($\pm \epsilon$)
