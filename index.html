Using Public Data to Visualize Changes in Home Prices in Orange County

This visualization uses public data to show how home prices that are located in zip code 92610 change over time and by geographic area. The first chart displays the median home price history for homes located in the same zip code, county, and metropolitan area alongside the national average. The second chart displays the average interest rate for a 30-year fixed mortgage. The third chart visualizes Zillow's forecasts for the change in home prices for each city located in Orange County.
